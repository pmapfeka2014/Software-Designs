README
--------------------------------------------------

1. Download the zip file prospermapfeka.zip
2. Double-click on the file Lab7_MemoryGame.html
3. Click new game to begin a new game. The game will not allow you to begin without clicking this button
4. Flip any two cards.If the cards are identical, they will vanish, if not they will flip back.Watch out for the clock.You have 30seconds to get it right. 
Every correct pair is 200points. 
5. When 6 pairs are flipped, the game is won 

Enjoy
